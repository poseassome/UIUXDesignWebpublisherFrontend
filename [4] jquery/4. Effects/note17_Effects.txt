*** 제이쿼리 효과

  **기본효과
    1. hide()
      - 지정된 요소를 감추기
      - 매개변수 없음
      - $("대상").hide()

    2. show()
      - 지정된 요소를 보이기
      - 매개변수 없음
      - $("대상").show()

    3. toggle()
      - 지정된 요소의 속성을 교차
      - $("대상").toggle()


=====
  fading <-- 결과적으로보면 basics인데 여기에 animation이 더해진다. (점점점 보이거나 점점점 사라지는 것, opacity)
  fadeTo ~~까지 투명해저라

  fadeIn는 필요인자 2개

    코드상에서 []는 순서, 배열의 개념
    var a = ["A", "B", "C"]
    document.write(a) = "A", "B", "C"
    document.write(a[0]) = "A"

  여기서 []는 생략가능한 인자 의미
  [duration]: 애니메이션이 진행되는 시간
  [complete]: 효과를 완료한 후에 실행할 문장
=====


  ** 페이드 효과
    1. fadeIn
      - 대상을 서서히 나타나게
      - $("대상").fadeIn( [ duration ] [ , complete ] )

    2. duration
      - 애니메이션이 진행되는 시간
      - String 타입, Number 타입
      - "slow", "fast", 1/1000s

    3. complete
      - function() 타입
      - 해당 대상이 효과를 완료했을 때 실행할 내용

    4. fadeOut
      - 대상을 서서히 사라지게
      - fadeIn과 사용방법 동일

    5. fadeTo
      - 지정된 opacity 값까지 fade
      - $("대상").fadeTo( duration, opacity [ , complete ] )
      - duration : fadeIn과 동일
      - opacity : Number 타입, 0~1 사이의 값만 사용
      - complete : fadeIn과 동일

    6. fadeToggle
      - 지정된 대상의 opacity를 변경
      - fadeToggle( [ duration ] [ , easing ] [ , complete ] )

  =====


  ** 슬라이드 효과
    1. slideDown
      - 미끄러지는 듯한 효과
      - slidDown( [ duration ] [ , complete ] )

    2. duration
      - 애니메이션이 진행되는 시간 (기본값:400)
      - String 타입, Number 타입
      - "slow', "fast", 1/1000s

    3. complete
      - function() 타입
      - 해당 대상이 효과를 완료했을 때 실행할 내용

    4. slideUp
      - slideUp( [ duration ] [ , complete ] )
      - slideDown과 사용방법 동일

    5. slideToggle
      - 지정된 요소의 slide 상태를 변경
      - slideToggle( [ duration ] [ , complete ] )

=====
  Basic, Fading, sliding은 다 같은 하나의 메소드로 만든 것이라,
  같이 사용하면 중복되어 동작하지 않음.

  easing : 가속되는 효과

  css메소드 사용법과 animate메소드 사용법은 동일하다.
=====

  * animate 효과
    1. animate
      - CSS로 제어 가능한 속성을 변화시키는 메소드
      - animate( properties [ , duration ] [ , easing ] [ , complete ] )
      - properties : CSS로 제어 가능한 속성과 값, 여러 개 사용 가능
      - duartion : Number 또는 String 타입
                   애니메이션 진행시간 (기본값:400)
      - easing : String 타입
                 가속효과 (기본값:swing)
      - complete : Function() 타입
                   효과가 완료 되었을 때 실행할 내용

    2. 이 외 메소드들은 animate을 지원해주는 메소드들
