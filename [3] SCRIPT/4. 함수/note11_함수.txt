** 함수 **
 미리 약속되어있는 명령어

  1. 내장 함수 - 만들어져있는 명령어
  2. 사용자 정의 함수 - 만들어서 사용하는 명령어

  예시] HTML, CSS에서와 똑같은 개념
    -태그 (재정의)
      a { color: red; }
    -사용자 정의
      . NAME {   }
      # NAME {   }


** 내장 함수 **
  1. 대화상자 함수
    - alert("문자열") : 메세지 상자
                        ✔ return value: 없음, 기본 인자 1개
    - confirm("문자열") : 확인 / 취소 선택 상자
                          return value(반환값-값이 되돌아오다)
                          창을 열어준 웹브라우저에게 자기가 뭐가 눌렸는지 알려준다.
                          ★반드시, 조건문과 같이 사용해야 한다. (true, false가 산출되니까.)
                          ✔ return value: T/F, 기본 인자 1개
    - prompt("질문메세지", "초기메세지") : 사용자 입력 상자
                                          ✔ return value: 사용자 입력값, 기본 인자 2개


** 사용자 정의 함수 **
  1. 사용자가 임의로 생성한 함수
  2. 선언문 : function      <-- 다른 많은 언어에서도 사용방법이나 의미가 똑같음
              명령어를 만드는 기능 (변수 선언은 var로 하듯이)
  3. function 함수명(매개변수, 매개변수...){
        실행문
        실행문
        .
        .
     }
  4. ⭐호출 시에만 실행
     (만들기만 한다고 실행되지않음)
  5. 사용자 정의 함수 호출방법
    - 단독 실행
    - 변수에 의한 실행
    - 다른 함수에 의한 실행
    - 이벤트에 의한 실행 (⭐⭐⭐⭐⭐/ 많이 사용)
  6. 함수명 규칙 : 변수명 규칙과 동일


  ** 이벤트 **
  프로그래밍 언어에서 이벤트란, trigger라는 표현을 쓰기도 함
  명령어들이 그냥 실행되는 게 아니라 내가 행동을 했을 때 실행됨(ex. 마우스를 올렸을 때, 닫기버튼을 눌렀을 때)

  사용자 정의 함수가 있다? 그럼 어딘가 이벤트가 있다. (소울메이트)

  javascript에서 이벤트는 <body>에서 적용할 수 있음

    1. 언제
    2. 이벤트 : 상황, ~~ 하다
    3. 이벤트 핸들러 : 이벤트 사용, ~~ 했을 때
                      이벤트 앞에 on 추가

      이벤트                이벤트 핸들러(trigger)
      load                  onload
      unload                onunload
      mouseover             onmouseover
      mouseout              onmouseout
      focus                 onfocus
      blur                  onblur
      click                 onclick
      mousedown             onmousedown
      mouseup               onmouseup
